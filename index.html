ECE Open Source Vending Machine

Also within the ECE Student Council, I am involved in a vending machine project that encourages users to contribute to open source code. In return, the user will obtain coins that allows them to activate and retrieve a "reward" from the vending machine that sits in the UCSD's Jacobs Hall. Besides the entire software project, we will also be building the vending machine from scratch. Within the project, we will be using the following skills: Python, Arduinos, Rasperry Pis, servers, databases, Github, web development, and Computer Aided Design.
